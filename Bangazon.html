What is Fitted?

Fitted is a clothing catalog and outfit creation app created to relieve the stress of getting dressed & helping you looking your best each morning. Easy to use Item and outfit views with filters by clothing type, outfit style, or individual items.

  • Utilized JavaScript, HTML, CSS, & Internal Database using JSON-Server
  • Learned Cloudinary for image storage andphoto upload
  • Clothing collection database and outfit creation application
  • full CRUD capabilities
  • Add/Edit/Remove Items and Outfits
  • Filter through items and outfits based on style, type of clothing, or item
  • Styled using HTML & CSS
