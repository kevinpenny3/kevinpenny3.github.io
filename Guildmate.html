GuildMate

What is GuildMate?

GuildMate is a World of Warcraft - Classic application allowing users to track their in game guild and events. Users can create their in game character, create or join a guild. Once that user is in a guild they will be able to see and sign up to attend their guilds events.

  • Utilized C#, .Net, SQL, Entity, MVC Pages
  • Authorization is handled through Identity Framework
  • World of Warcraft - Classic guild organization and calendar application
  • full CRUD capabilities
  • Add/Edit/Remove Guild Events, Guild Members
  • Filter through Servers and Guild Listings
  • Styled using Bootstrap and CSS
