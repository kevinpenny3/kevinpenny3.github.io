Holiday Road

What is Holiday Road?

Holiday Road is a travel planning app allowing a user to select State Parks from a third party API. Once they have chosen a park they can select other attractions, restaurants and save the itinerary. Users will also see a 7 day forecast for their chosen location

  • Worked in a group environment for development while focusing on individual components
  • Individually responsible for Javascript functionality using API calls, user selects, saving, and displaying selections on the DOM
  • Took lead on CSS styling based on persona
